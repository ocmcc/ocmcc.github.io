摘要

  1. 整数类型的无限范围及4种进制表示
  2. 浮点数类型的近似无限范围、小尾数及科学计数法
  3. +、-、*、/、//、%、**、二元增强赋值操作符
  4. abs()、divmod()、pow()、round()、max()、min()

整数类型

可正可负，没有取值范围限制

pow(x,y)函数：计算xy

十进制：1010，99，-217

二进制：0b或0B开头，0b010，-0B101 b - binary

八进制: 0o或0O开头，0o123，-0O456 o - octal

十六进制：0x或0X开头，0x9a，-0X89 hex

浮点类型

浮点数取值范围和小数精度都存在限制，但常规计算可忽略

取值范围数量级约-10308至10308，精度数量级10-16,因此存在不确定尾数

比较浮点数大小时，用round()函数。

round(x,d)：对x四舍五入，d是小数截取位数，Python3的round()函数四舍五入取整时，采用最近偶数原则

数值运算操作符

  1. x/y（除，产生浮点数结果，和c语言不一样）与x//y（整数除）的区别

  2. x%y 余数，模运算，10%3结果为一

  3. x**y 幂运算，x的y次幂

（当y是小数时，开方运算，10**0.5的结果是√10）

数字类型的关系

类型间可以混合运算，生成结果为“最宽”类型

整数→浮点数→复数

整数 + 浮点数 = 浮点数，结果的结尾就变成不确定尾数

数值运算函数

divmod(x,y) 商余，（x//y,x%y）,同时输出商和余数。

pow(x,y[,z])幂余，(x**y)%z，[..]表示参数z可省略
